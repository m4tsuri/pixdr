# Runtime UHD images

This directory is generated by `scripts/fetch-uhd-images.sh`.
Only these B200/B210 files are packaged into the APK:

- `usrp_b200_fw.hex`
- `usrp_b200_fpga.bin`
- `usrp_b210_fpga.bin`

The binary image files are ignored by git. Check Ettus/UHD licensing before
redistributing them as repository contents or release assets.
